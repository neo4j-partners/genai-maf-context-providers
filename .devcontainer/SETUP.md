# Setup

This repository accompanies the [Neo4j Context Providers for Microsoft Agent Framework course](https://graphacademy.neo4j.com/courses/genai-maf-context-providers/) on [GraphAcademy](https://graphacademy.neo4j.com).

When the devcontainer is created, all required software and packages are installed.

Follow the [Setup Instructions in GraphAcademy](https://graphacademy.neo4j.com/courses/genai-maf-context-providers/1-introduction/2-setup/) to get started.

You will need to:

1. Create a new `.env` file and copy contents of `.env.example`
2. Update environment values with the [Setup Instructions](https://graphacademy.neo4j.com/courses/genai-maf-context-providers/1-introduction/2-setup/)
3. Run `genai-maf-context-providers/test_environment.py` to verify setup
